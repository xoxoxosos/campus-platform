## 四、配置管理设计

### 4.1 配置分层原则

```
application-common.yml        ← 公共参数（驱动类、连接池、JPA方言、Jackson）
    ↓ 被 application.yml 引入
application.yml               ← 模块独有配置（端口、JWT、短信配置）
    ↓ 激活 profile
application-dev.yml           ← 环境具体连接信息（数据库URL、Redis地址）
```

### 4.2 配置内容分工

| 文件 | 位置 | 内容 |
|------|------|------|
| `application-common.yml` | campus-common | 数据库连接池、JPA方言、Redis连接池、Jackson |
| `application.yml` | campus-login | 端口、JWT参数、短信参数、`spring.config.import` |
| `application-dev.yml` | campus-login | MySQL URL/用户名/密码、Redis host/port |
| `application-prod.yml` | campus-login | 环境变量占位符 `${DB_URL}` 等 |
